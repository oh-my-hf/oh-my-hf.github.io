# ohmyhf website

Official landing page for **ohmyhf (Oh My HuggingFace)** — an unofficial, open-source,
cross-platform Hugging Face desktop client.

> This repository is being bootstrapped. The site (Astro + TypeScript + Tailwind CSS,
> static output, English + Simplified Chinese) is built via orchestrated cloud agents.

## Status

Initial scaffold commit. Full site, tooling, and docs land in subsequent commits.
